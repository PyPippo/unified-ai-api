# Examples

This document provides an overview of the examples included with the Unified AI API connection library.

## Available Examples

### Basic Usage (`examples/basic_usage.py`)

A simple interactive chat interface demonstrating:

- Basic API connection setup
- Simple conversation flow
- Error handling
- Clean exit procedures

**What you'll learn:**

- How to create an `APIConnectionManager`
- Basic interactive chat implementation
- Simple error handling patterns

### Advanced Usage (`examples/advanced_usage.py`)

A comprehensive example showcasing advanced features:

- Multi-session management
- Interactive configuration setup
- Programmatic configuration
- Advanced error handling
- Session persistence

**What you'll learn:**

- Advanced configuration patterns
- Multi-session handling
- Complex error recovery
- Configuration persistence

### OpenAI Example (`examples/openai_example.py`)

Specific examples for OpenAI-compatible APIs:

- OpenAI API configuration
- Model selection
- Parameter tuning
- Streaming responses

**What you'll learn:**

- OpenAI-specific configuration
- Model parameter optimization
- Streaming response handling

### REST API Example (`examples/rest_api_example.py`)

Examples for generic REST API integration:

- Custom endpoint configuration
- Authentication handling
- Request/response processing
- Custom headers and parameters

**What you'll learn:**

- Generic REST API integration
- Custom authentication methods
- Request customization
